CGPA & GPA Calculator
A simple and accurate application to calculate GPA for each semester and overall CGPA using credit and grade inputs.
Features
GPA Calculator: Enter subject credits and grades to get GPA
CGPA Calculator: Add multiple semester GPAs to compute CGPA
Two separate pages: One for GPA and one for CGPA
Supports O, A+, A, B+, B, C grade scales
Clean and simple UI
How GPA Is Calculated
GPA = (Sum of (Credit × Grade Point)) / (Total Credits)
How CGPA Is Calculated
CGPA = (Sum of (Semester GPA × Semester Credits)) / (Total Credits)
Grade Points
O = 10
A+ = 9
A = 8
B+ = 7
B = 6
C = 5
How to Use
Open the application
Select GPA or CGPA
For GPA: Enter credits and grades, then calculate
For CGPA: Enter semester GPAs and credits, then compute
